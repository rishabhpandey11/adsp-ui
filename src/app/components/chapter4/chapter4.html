2. Theory Summary

2.1 What is Quantization?

Quantization is the process of rounding a continuous-amplitude signal to a finite set of discrete levels so that it can be represented digitally.

When an analog signal x(t) is sampled ‚Üí x[n], its amplitude remains continuous.

Quantization replaces each sample with the nearest discrete level qk from a finite set of L = 2NL possible values.
