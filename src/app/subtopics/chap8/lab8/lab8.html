Apply filter design techniques and multirate processing concepts through hands-on Python exercises. You will design FIR filters using the Parks-McLellan algorithm, implement Noble Identities, and build efficient polyphase sampling rate converters.

Exercise 1: FIR Filter Design with remez

Design a half-band low-pass filter using the Parks-McLellan algorithm and explore the effect of filter length and weights

Task: Design a half-band low-pass filter with pass band up to 0.4π and stop band from 0.5π (transition band: 0.4π – 0.5π).

Questions:

1. Is the phase linear? How can you tell from the plot?

2.What happens to stop band attenuation if you change the weight to W = [1, 1000]?

3.Try increasing N to 64. What improves?

4.What is the relationship between the stop band weight and attenuation?
