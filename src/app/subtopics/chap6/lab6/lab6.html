Chapter 6 : Lab

Objective

Apply sampling, downsampling, and upsampling concepts through hands-on Python exercises. You will implement resampling operations, analyze aliasing and imaging effects, and visualize their behavior in time and frequency domains.

Exercise 1: Upsampling and Interpolation

Goal: Understand that upsampling creates spectral images that must be removed by interpolation filtering.

Task:

  • Observe the spectral images created by zero-insertion upsampling.
  • Design an anti-imaging lowpass filter.
  • Compare the spectrum before and after interpolation filtering.

Questions:

1. What happens if you forget the factor of L in the interpolation filter output?

2. Try different values of L (2, 3, 8). How does this affect the spacing of spectral images?

Exercise 2: Combined Resampling (Rational Rate Conversion)

Goal: Implement rational rate conversion L/M by combining upsampling and downsampling.

Task:

  • Upsample by L, then downsample by M to change sampling rate by factor L/M.
  • Use a single anti-aliasing/anti-imaging filter.
  • Compare with scipy's resample_poly function.

Questions:

1. What happens to frequencies above the new Nyquist frequency?

2. Why is resample_poly more efficient than the manual approach?

3. Try resampling from 44.1 kHz to 48 kHz (ratio 160/147). What L and M values would you use?

Exercise 3: Real-time Sampling Visualization

Goal: Create an interactive visualization showing time and frequency effects of downsampling.

Task:

  • Visualize a signal being downsampled in real-time.
  • See the aliasing components appear in the spectrum.
  • Observe the effect of anti-aliasing filters.

Questions:

1. At which value of N do you first observe significant aliasing for the 3500 Hz component

2. How would you modify the signal to prevent aliasing at N=8?

3.Can you add an anti-aliasing filter to the animation?
