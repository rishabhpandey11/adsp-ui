Chapter 4 : Lab

Objective

Apply Lloyd-Max quantization and iterative optimization concepts through hands-on Python exercises. You will implement the Lloyd-Max algorithm, compare it with uniform quantizers, and analyze quantization error for different probability distributions.

Exercise 1: Understanding Quantization Basics

Goal : Implement a simple uniform quantizer and visualize quantization error.

Task: Implement both quantizer types and compare their behavior for signals centered around zero.

  • Generate a signal with known distribution
  • Apply uniform quantization with different numbers of levels
  • Compute and visualize the quantization error

Questions:

1. How does MSE change with the number of quantization levels?

2. For uniform distribution, are the reconstruction levels equally spaced?

3. What is the relationship between MSE and $M$?

Exercise 2: Lloyd-Max Algorithm Implementation

Goal: Implement the core Lloyd-Max iterative algorithm for a given PDF.

Task: Implement the two-step iterative algorithm: nearest neighbor rule and centroid condition.

Questions:

1. Does the algorithm converge to the expected solution for uniform distribution?

2. How many iterations does it typically take?

3. What happens if you use different random initializations?

Exercise 3: Gaussian Distribution Analysis

Goal: Design Lloyd-Max quantizer for Gaussian distributed signals.

Task: Compare uniform and Lloyd-Max quantizers for zero-mean Gaussian signals.

Questions:

1. For Gaussian signals, where are reconstruction levels most densely placed?

2. How much SNR improvement does Lloyd-Max provide over uniform quantization?

3. What happens as $M$ increases?
