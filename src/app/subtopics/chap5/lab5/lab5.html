Chapter 5 : Lab

Objective

Apply vector quantization and LBG algorithm concepts through hands-on Python exercises. You will implement VQ encoders/decoders, train codebooks using the LBG algorithm, and analyze distortion vs. codebook size trade-offs.

Exercise 1: Fixed Codebook Vector Quantization

Goal: Understand the basic encoding and decoding process with a pre-defined codebook.

  • Change the CODEBOOK array and re-run
  • Change N_SAMPLES and the data distribution:

Questions:

1. How does distortion change as you increase the number of codewords?

2. What happens when codeword placement is poor (e.g., all in one corner)?

Exercise 2: Implementing the LBG Algorithm

Goal: Implement the iterative LBG algorithm to train a codebook from scratch.

Task:

  • Implement the two key steps: nearest-neighbor assignment and centroid update.
  • Monitor convergence by tracking distortion over iterations.
  • Visualize how codewords move during training.

Questions:

1. How many iterations does it take to converge?

2. What happens if you initialize with very poor codeword positions?

3. Try different values of K (2, 4, 8). How does convergence speed change?

Exercise 3: Codebook Size vs. Distortion Trade-off

Goal: Analyze the rate-distortion trade-off by varying codebook size.

Task:

  • Train codebooks of different sizes (K = 2, 4, 8, 16, 32).
  • Plot distortion vs. codebook size.
  • Plot bits-per-vector vs. distortion.

Questions:

1. At what point does adding more codewords give diminishing returns?

2. If you had a bit-rate budget of 4 bits per vector, which K would you choose?

3.How does the distortion reduction scale with doubling K?

Exercise 4: Speech Signal Vector Quantization

Goal: Apply VQ to a real 1D signal by forming vectors from consecutive samples.

Task:

  • Create vectors of dimension N=4 from a speech-like signal.
  • Train a codebook and encode/decode the signal.
  • Compare original vs. reconstructed signal and compute SNR.

Questions:

1. How does SNR change with different K values (try 8, 16, 64)?

2. What happens if you change VECTOR_DIM to 2 or 8?

3. Can you hear the difference if you save and play the audio?
